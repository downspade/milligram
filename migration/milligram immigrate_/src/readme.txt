【始めに】

 SusiePluginを読み込み可能にするNiVE用プラグインです。

 ちょこまかいじってたら1,4,8bppの画像も読み込めるようになったｗｗｗ

 ちなみにARGB32bpp(透過情報混)の画像は基本的には対応してますが、
 Spi側（SusiePluginのほう）が対応してないと読み込めません。
 // Spiの殆どが対応してないと思う。


 このプラグイン(NiVE側)でエラーが発生してもSpi（SusiePluginのほう）作者方に「ここがおかしい！」
 といった連絡をしないでください。大変迷惑になります。


 本体(BxLib.dll)から分離して、ソースを付けて公開しました。

 著作権は放棄するので自由に改変してください。
